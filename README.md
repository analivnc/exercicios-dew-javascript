# 💻 JavaScript - Exercícios de Desenvolvimento Web

Olá! 👋

Este repositório reúne os exercícios que estou desenvolvendo durante a disciplina de **Desenvolvimento Web** da **Univille**.

O objetivo é praticar JavaScript, desenvolver a lógica de programação e acompanhar minha evolução ao longo da matéria.

## 📚 O que estou estudando

- Variáveis (`let` e `const`)
- Operadores
- Estruturas condicionais (`if` / `else`)
- Laços de repetição (`for`)
- Arrays
- Strings
- Funções
- Template Literals
- Métodos como:
  - `split()`
  - `join()`
  - `push()`
  - `includes()`

## 📂 Organização

Cada pasta ou arquivo corresponde a um exercício proposto em aula.

Conforme novos conteúdos forem sendo apresentados, este repositório será atualizado.

## 🚀 Objetivo

Mais do que apenas entregar atividades, este repositório representa meu processo de aprendizagem em JavaScript. Aqui estão registrados meus estudos, desafios e a evolução da minha lógica de programação.

> "Todo especialista já foi iniciante um dia." ✨
